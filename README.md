# Next.js & HeroUI & DrizzleORM & Neon Database Template

## Technologies Used

- Next.js
- HeroUI
- Next Auth
- DrizzleORM
- Neon Database

## How to Use

### Install dependencies

```bash
pnpm install
```

### Run the development server

```bash
pnpm run dev
```
